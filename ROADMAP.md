# Nova Diagnostics Roadmap

## Phase 1: Current marketing website and booking flow

- Mobile-first marketing website for Nova Diagnostics in Vashi.
- Test catalogue, health packages, home collection flow, prescription request UI, contact and local SEO pages.
- WhatsApp, call, directions and booking CTAs.
- Config-driven business details and editable test/package data.

## Phase 2: Basic backend for bookings and prescription upload

- Secure booking request API.
- Private prescription upload storage.
- Staff email/WhatsApp notifications.
- Basic spam protection and consent logging.

## Phase 3: LIMS integration or manual staff dashboard

- Internal staff dashboard for request triage.
- Manual status tracking for bookings.
- Optional LIMS integration if the lab adopts compatible software.

## Phase 4: OTP-secured report download

- Patient report access by verified mobile number.
- OTP-based authentication.
- Expiring download links and audit logs.

## Phase 5: Doctor referral management

- Referral request tracking.
- Clinic pickup coordination.
- Doctor-facing communication workflows without exposing unnecessary patient data.

## Phase 6: Corporate health checkup lead management

- Corporate inquiry forms.
- Lead qualification workflow.
- Package proposal tracking.

## Phase 7: Advanced analytics and automation

- Conversion analytics for calls, WhatsApp, forms and directions.
- Follow-up automation for unconfirmed bookings.
- Search analytics for popular local tests and packages.
